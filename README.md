## For the end user

Run your clinic's finances and compliance from one HTML file. 9 tabs cover appointments, revenue, prescribing, staff, CQC, expenses, and practice P&L.

### Key features

| Tab | What it does |
|---|---|
| Dashboard | KPI overview — revenue, appointments, DNAs, prescribing |
| Appointments | Appointment log with DNA tracking and auto-revenue |
| Revenue | NHS and private income streams |
| Prescribing | Drug costs per item and prescriber |
| Staff & CPD | GMC/NMC registration, CPD hours, appraisal dates, DBS |
| CQC | 5 key questions checklist + regulatory requirements |
| Practice P&L | Rolling 12-month breakdown with overheads |

### Key rules

- **CQC**: Safe, Effective, Caring, Responsive, Well-led (annual fee ~£4,428)
- **GMC revalidation**: 5-year cycle, annual appraisals
- **Medical indemnity**: mandatory for all clinical staff
- **Prescribing**: local formulary, NICE TAs mandatory
- **Duty of candour**: Regulation 20, HSCA 2008
